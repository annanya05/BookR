How to run this app?
Clone this repository: git clone https://github.com/annanya05/login.git && cd login/
Activate your python veritualenv: 2.1 Creating a new virtualenv: virtualenv venv 2.2 Activating this env: source venv/bin/activate
Install app dependencies in this virtualenv: pip install flask
Set FLASK_APP in your environment variable: export FLASK_APP=app.py
Run your app: flask run
